About

This project aims to utilize Deep Neural Networks (DNNs) to classify different land types, such as agriculture, water bodies, urban areas, deserts, roads, and trees, using satellite imagery captured by the Sentinel-2 satellite. Sentinel-2 is part of the Copernicus program by the European Space Agency (ESA) and provides high-resolution multispectral images, making it ideal for land-use classification applications. By leveraging this data, we can analyze both natural and urban environments with high precision, enabling informed decision-making in various sectors. The goal of this project is to develop an AI-powered model capable of accurately classifying geographic areas, which can be applied in several critical fields such as urban planning, forest monitoring, natural resource assessment, and disaster response. Additionally, the model can be used to monitor environmental changes over time, such as desertification or urban expansion, providing valuable insights for policymakers and environmental researchers. To achieve these objectives, the project will utilize open-source datasets or generate custom datasets using tools like QGIS. The data will undergo preprocessing and enhancement before being fed into the deep neural network. By refining the model and incorporating advanced deep-learning techniques, the classification accuracy will be improved, enhancing its practical usability across various applications.
